# ENUM

### **enumeration type**

관련 있는 상수들의 집합. 코드값처럼 서로 관련이 있고, 발생할 항목이 뻔한 대상을 목록화 해서 관리.

코드값이 1이라고 다 같은 1이 아님. => 각각의 의미에 맞게 구분하려고 씀.&#x20;

* 값 자체를 사용할 수도 있고 `value()`&#x20;
* 함수처럼 각 항목에 대응되는 값을 활용할 수도 있고 `valueOf()`&#x20;
* 인덱스를 활용할 수 있음 `ordinal()`
